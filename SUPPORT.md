# Support

The scripts, samples, and tools in this repository are community-supported sample code provided as-is, without a service-level agreement.

## Get help

1. Review the resource README and search [existing GitHub issues](https://github.com/microsoft/FastTrack/issues).
2. If the problem concerns a repository sample, [open a bug report](https://github.com/microsoft/FastTrack/issues/new/choose) with reproducible steps and sanitized logs.
3. For product break/fix support, use the support channel included with your Microsoft 365 subscription or visit [Microsoft 365 support](https://support.microsoft.com/microsoft-365).
4. For Microsoft FastTrack engagement questions, visit [Microsoft FastTrack](https://www.microsoft.com/fasttrack).

Do not include credentials, tenant data, personal information, or other sensitive content in an issue. Report security vulnerabilities according to [SECURITY.MD](SECURITY.MD), not through a public issue.
